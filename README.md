# Portfolio

Short personal portfolio built with React, Vite and Tailwind CSS showcasing projects, skills and contact information.

Live demo: https://portfolio-six-sable-wvf2003v47.vercel.app/  

## Quick Start

- Install: `npm install`
- Run locally: `npm run dev`
- Build: `npm run build`
- Preview build: `npm run preview`

## Tech

- Vite
- React
- Tailwind CSS
